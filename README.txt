HELLO!

This the keyboard playable game not run by an AI
please use 'a' and 'd' to move left and right respectively
and 'SPACE' to jump!

Please run 'main.py' as the executable for this game to play it!

Note:
This was run on Python 3.10 64 bit but should work on any instance of python with PyGame installed!
